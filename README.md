# Dpka's Portfolio

My personal portfolio website — built to showcase my work as a developer focused on backend systems, full-stack development, and AI integrations.

## Architecture & Design
Built using a custom **Neo-Brutalist** design system. Explores modern web boundaries through heavy sticky-stacking full-viewport sections, deeply integrated scaling typography, Intersection Observer physics, and aggressive raw-color highlighting.

## Tech Stack
- Next.js 15 (App Router)
- React 19
- Tailwind CSS v4
- TypeScript

## Local Development
```bash
npm install
npm run dev
```
